## Description

[Bot](https://t.me/nest_links_bot) on Telegram for working with URL's.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
