## Description

[Bot](https://t.me/nest_links_bot) on Telegram for working with URL's.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## About

Deployed on "Render" free-instance, may take a few minites before first answer [read more here](https://docs.render.com/free).
